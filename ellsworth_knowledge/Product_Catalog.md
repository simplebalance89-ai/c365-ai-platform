## TECHNICAL SELECTION PARAMETERS

### Cure Types
| Type | Description | Typical Use |
|---|---|---|
| UV/Light Cure | Cures on UV/visible light exposure | High-speed production, electronics |
| Heat Cure | Requires 150-350°F | Aerospace, structural, high-performance |
| Room Temp (RTV) | Cures at ambient | Sealing, general assembly |
| Moisture Cure | Atmospheric humidity (10s-2min green strength) | Silicones, some urethanes |
| Anaerobic | Cures in absence of air (metal-to-metal) | Threadlocking, retaining, gasketing |
| Cyanoacrylate | Instant cure from surface moisture | Fast bonding, small parts |
| Two-Part Mix | Mixing resin + hardener | Epoxies, urethanes, structural |

### Key Physical Properties
| Parameter | Range | Why It Matters |
|---|---|---|
| Viscosity (cP) | 50 to 100,000+ | Flow, dispensability, gap fill |
| Shear Strength (psi) | 500 to 4,350+ | Bond strength under shear |
| Temperature Range | -65°F to 500°F+ | Operating range of cured bond |
| Cure Time | Seconds to 24+ hours | Processing time |
| Glass Transition (Tg) | Varies | Thermal cycling behavior |
| Hardness (Shore A/D) | Shore 00 (gel) to Shore D (rigid) | Flexibility vs rigidity |
| Thermal Conductivity (W/mK) | 0.2 to 6+ | Heat transfer capability |

### Substrate Compatibility
| Substrate | Best Adhesive Types |
|-----------|---------------------|
| Aluminum | Epoxy, Acrylic, Urethane |
| Steel | Epoxy, Acrylic, Anaerobic |
| Stainless Steel | Epoxy (with primer), Acrylic |
| Carbon Fiber | Epoxy (film or paste) |
| Plastics (ABS, PC) | Acrylic, Cyanoacrylate |
| Polyethylene/PP | Surface treatment + Acrylic |
| Glass | UV Curable, Silicone, Epoxy |
| Rubber | Cyanoacrylate, Urethane |

### Temperature Guide
| Service Temp | Recommended Types |
|---|---|
| Below -40°F | Flexible epoxy, Silicone |
| -40°F to 250°F | Standard epoxies |
| 250°F to 400°F | High-temp epoxy, Silicone |
| Above 400°F | Ceramic adhesives, Inorganic |
